# Scientific Computing
 
This is a repository of simple programs to learn basic concepts of parallel computing in a computer-science course, CSCI 596 (Scientific Computing and Visualization) at Unversity of Southern California (USC):
https://aiichironakano.github.io/cs596.html

## Parallel Molecular Dynamics
Folder pmd: Parallel molecular dynamics (MD) using message passing interface (MPI)

## Hybrid Parallel Molecular Dynamics

Folder hmd: Hybrid parallel MD combining MPI with OpenMP for multithreading
